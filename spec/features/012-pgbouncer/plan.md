# 012 - PgBouncer

## Objetivo

Implementar PgBouncer como administrador de conexiones entre el backend del proyecto Pacific Control y la base de datos PostgreSQL.

## Alcance

Este feature comprende:

- Instalación y configuración de PgBouncer.
- Configuración de la conexión con PostgreSQL.
- Configuración de la conexión del backend mediante PgBouncer.
- Definición de parámetros de conexiones.
- Verificación del funcionamiento de las conexiones.

## Dependencias

- Backend Base
- PostgreSQL
- PgBouncer
- SQLAlchemy

## Resultado esperado

El sistema contará con un administrador de conexiones configurado para centralizar el acceso del backend a PostgreSQL.
