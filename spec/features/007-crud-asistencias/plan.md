# 007 - CRUD Asistencias

## Objetivo

Implementar el módulo de gestión de asistencias del proyecto Pacific Control, permitiendo registrar, consultar, actualizar y administrar los registros de presencia del personal operativo mediante una API REST.

## Alcance

Este feature comprende:

- Registro de asistencias.
- Consulta de asistencias.
- Actualización de información.
- Cambio de estado de la asistencia.
- Asociación de asistencias con empleados, turnos y dispositivos.

## Dependencias

- Backend Base
- PostgreSQL
- SQLAlchemy
- API REST
- CRUD Empleados
- CRUD Turnos
- CRUD Dispositivos

## Resultado esperado

El sistema permitirá administrar los registros de asistencia mediante operaciones CRUD, garantizando la integridad y consistencia de los datos.
