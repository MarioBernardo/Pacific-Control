# 004 - CRUD Puestos

## Objetivo

Implementar el módulo de gestión de puestos del proyecto Pacific Control, permitiendo registrar, consultar, actualizar y administrar la información de los puestos asignados al personal operativo mediante una API REST.

## Alcance

Este feature comprende:

- Registro de puestos.
- Consulta de puestos.
- Actualización de información.
- Cambio de estado del puesto.
- Eliminación lógica de registros, cuando corresponda.

## Dependencias

- Backend Base
- PostgreSQL
- SQLAlchemy
- API REST

## Resultado esperado

El sistema permitirá administrar la información de los puestos mediante operaciones CRUD, garantizando la integridad y consistencia de los datos.
