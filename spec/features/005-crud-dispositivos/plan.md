# 005 - CRUD Dispositivos

## Objetivo

Implementar el módulo de gestión de dispositivos del proyecto Pacific Control, permitiendo registrar, consultar, actualizar y administrar la información de los dispositivos asignados a los puestos mediante una API REST.

## Alcance

Este feature comprende:

- Registro de dispositivos.
- Consulta de dispositivos.
- Actualización de información.
- Cambio de estado del dispositivo.
- Asociación de dispositivos con puestos.

## Dependencias

- Backend Base
- PostgreSQL
- SQLAlchemy
- API REST
- CRUD Puestos

## Resultado esperado

El sistema permitirá administrar la información de los dispositivos mediante operaciones CRUD, garantizando la integridad y consistencia de los datos.
