# 008 - CRUD Novedades

## Objetivo

Implementar el módulo de gestión de novedades del proyecto Pacific Control, permitiendo registrar, consultar, actualizar y administrar las incidencias reportadas durante los turnos mediante una API REST.

## Alcance

Este feature comprende:

- Registro de novedades.
- Consulta de novedades.
- Actualización de información.
- Cambio de estado de la novedad.
- Asociación de novedades con empleados y turnos.

## Dependencias

- Backend Base
- PostgreSQL
- SQLAlchemy
- API REST
- CRUD Empleados
- CRUD Turnos

## Resultado esperado

El sistema permitirá administrar las novedades mediante operaciones CRUD, garantizando la integridad y consistencia de los datos.
