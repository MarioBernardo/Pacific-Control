# 003 - CRUD Empleados

## Objetivo

Implementar el módulo de gestión de empleados del proyecto Pacific Control, permitiendo registrar, consultar, actualizar y administrar la información del personal de seguridad mediante una API REST.

## Alcance

Este feature comprende:

- Registro de empleados.
- Consulta de empleados.
- Actualización de información.
- Cambio de estado del empleado.
- Eliminación lógica de registros, cuando corresponda.

## Dependencias

- Backend Base
- PostgreSQL
- SQLAlchemy
- API REST

## Resultado esperado

El sistema permitirá administrar la información de los empleados mediante operaciones CRUD, garantizando la integridad y consistencia de los datos.
