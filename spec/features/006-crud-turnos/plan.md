# 006 - CRUD Turnos

## Objetivo

Implementar el módulo de gestión de turnos del proyecto Pacific Control, permitiendo registrar, consultar, actualizar y administrar la información de los turnos asignados al personal operativo mediante una API REST.

## Alcance

Este feature comprende:

- Registro de turnos.
- Consulta de turnos.
- Actualización de información.
- Cambio de estado del turno.
- Asociación de turnos con empleados y puestos.

## Dependencias

- Backend Base
- PostgreSQL
- SQLAlchemy
- API REST
- CRUD Empleados
- CRUD Puestos

## Resultado esperado

El sistema permitirá administrar la información de los turnos mediante operaciones CRUD, garantizando la integridad y consistencia de los datos.
