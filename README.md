# Pacific Control

## Descripción

Pacific Control es un sistema para la gestión de asistencia, novedades y turnos del personal operativo mediante una aplicación móvil y una API REST.

## Estado del proyecto

🚧 En desarrollo

## Tecnologías

- Python
- Flask
- PostgreSQL
- Redis
- Celery
- Flutter
- Git
- GitHub

## Características

- Gestión de empleados
- Gestión de puestos
- Gestión de dispositivos
- Gestión de turnos
- Registro de asistencias
- Registro de novedades
- API REST desarrollada con Flask
- Caché implementada con Redis (Cache Aside)
- Procesamiento asíncrono mediante Celery
- Configuración de infraestructura para autenticación JWT
- Base de datos PostgreSQL
