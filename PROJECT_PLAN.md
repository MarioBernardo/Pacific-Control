# Pacific Control

## Objetivo General

Desarrollar una aplicación móvil y una API REST para el control de asistencia, novedades y turnos del personal operativo de una empresa de seguridad privada.

## Objetivos específicos

- Gestionar empleados.
- Gestionar puestos.
- Registrar asistencia.
- Registrar novedades.
- Administrar turnos.
- Consumir la API desde Flutter.
- Optimizar el backend mediante Redis y Celery.

Backend
